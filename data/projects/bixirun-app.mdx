## Обзор

Личный проект по созданию мобильного приложения с нуля, демонстрирующий навыки full-stack разработки и управления продуктом.

## Технический стек

- **Frontend:** React Native, TypeScript, Expo
- **Backend:** Supabase (PostgreSQL, Auth, Realtime)
- **Платформы:** iOS (TestFlight), Android

## Функционал

- **Архитектура:** Компонентная архитектура с кастомными хуками и стейт-менеджментом.
- **UX/UI:** Современный адаптивный интерфейс, спроектированный по принципам mobile-first.
- **Синхронизация:** Синхронизация данных в реальном времени через Supabase.

## Статус

Приложение доступно для внутреннего тестирования через **TestFlight**. Проект служит доказательством концепции быстрой мобильной разработки с использованием современных кросс-платформенных инструментов.
